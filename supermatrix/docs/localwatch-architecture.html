双层防御

防御层 职责 来源
主进程内部 Migration idempotency、optional 降级、announce 警告、ProcessLifecycle 优雅重启 migration-resilience-design.md
Watchdog 外部 进程保活、crash loop 熔断、schema 漂移修复、飞书连通性、Scheduler 保活 localwatch
两层独立运作。主进程内部防御失效（如今天 migration 008 事故），watchdog 外部兜底。反之亦然。
